How to play:

1. http://ec2-52-50-103-145.eu-west-1.compute.amazonaws.com:8080/  (or localhost:8080 if you deploy it locally)          
2. player one create new game (push «New Game» button)
3. player two connect to the exist game using gameId from first player and field «Join Game»
4. use buttons «House 1, 2, 3, …» for make your turn
5. use «Refresh Game» button to see who is next (Is Your Turn: true - you can turn now)
